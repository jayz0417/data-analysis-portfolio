## Elecle 개요
쏘카에서 목말랐던 데이터분석을 공부하고 SQL을 이용한 분석과 BI 툴을 주로 사용하며 데이터분석의 역량을 다듬은 데이터분석의 첫 커리어 출발점    
```
특히 SQL의 여러 기법을 이용하여 테이블 조인, 데이터마트 기획한 경험과 BI 툴(Holistics)을 이용한 대시보드, 정기 보고서 등을 기획하여 지표를 관리했던 경험을 쌓을 수 있었음
```
